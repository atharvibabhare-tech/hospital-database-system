#  Hospital Database Management System

A structured MySQL-based Hospital Database project demonstrating database design, table relationships, and advanced SQL queries.

---

##  Database Structure

This project contains:

- Doctors Table
- Patients Table
- Medicines Table
- Appointments Table
- Prescriptions Table
- Surgical Items Table

---

##  SQL Concepts Used

- CREATE & ALTER
- INSERT
- SELECT
- WHERE
- LIKE
- ORDER BY
- GROUP BY
- HAVING
- JOIN
- Aggregate Functions (COUNT, AVG)

---

##  Example Queries

- Find medicines containing 'in'
- Count doctors per specialization
- Show patient & doctor details for appointments
- Average medicine price per category
- Latest 5 appointments
- Doctors with more than 2 appointments

---

##  Technology Used

- MySQL
- SQL

---

##  How to Run

1. Open MySQL Workbench
2. Run `hospital_db_project.sql`
3. Execute queries
